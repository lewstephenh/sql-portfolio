## Description
Developed SQL workflow to transform and analyze synthetic healthcare claims data. Project integrates ICD-10 diagnostic descriptions into raw claims via table joins, enhancing data interpretability. Calculates insurance-level claim denial rates using conditional logic and aggregation, enabling comparative performance analysis across payers. Additional steps include filtering claims for specific diagnoses, sorting records for reporting purposes, and creating streamlined tables with selected fields to support downstream analytics. Work demonstrates practical experience with SQL joins, CTEs, conditional expressions, aggregation, filtering, and data preparation for healthcare analytics.
